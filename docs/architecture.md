# Architektur – PDD Meta Evaluator & Mainstream Prompt Library

## Module

- loader.py – lädt Prompts
- evaluator.py – bewertet LLM-Antworten
- metrics.py – erzeugt Scores
- drift_detector.py – erkennt Drift
- hallucination_detector.py – erkennt Halluzinationen
- agent.py – orchestriert den Evaluationsprozess

## Datenfluss

1. Prompt wird geladen
2. LLM wird ausgeführt
3. Evaluator bewertet Antwort
4. Metrics Engine erzeugt Scores
5. Drift Detector prüft Konsistenz
6. Halluzination Detector prüft Struktur
7. Agent orchestriert alles
