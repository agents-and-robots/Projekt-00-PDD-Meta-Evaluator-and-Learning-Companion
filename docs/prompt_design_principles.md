# Prompt Design Principles

1. Jeder Prompt ist kurz, klar und eindeutig.
2. Jeder Prompt enthält eine Dokumentation, warum er so aufgebaut ist.
3. Jeder Prompt ist LLM-ready.
4. Jeder Prompt ist deterministisch.
5. Jeder Prompt vermeidet Halluzinationsanreize.
6. Jeder Prompt ist modular und austauschbar.
7. Jeder Prompt ist wissenschaftlich motiviert.
