# Drift-Check-Prompt

## Prompt
"Vergleiche Projekt A und Projekt B. Wo gibt es strukturellen, semantischen oder methodischen Drift?"

## Warum dieser Prompt so aufgebaut ist
- Drift ist ein Kernproblem in LLM-generierten Projekten
- Dieser Prompt macht Drift sichtbar

## Erwartetes Verhalten
- konkrete Unterschiede
- keine erfundenen Dateien
