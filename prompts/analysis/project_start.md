# Projekt-Start-Prompt

## Prompt
"Lies diese instructions.md vollständig. Welche Dateien würdest du erzeugen und in welcher Reihenfolge?"

## Warum dieser Prompt so aufgebaut ist
- Testet Strukturverständnis
- Prüft, ob das LLM die Projektarchitektur erkennt
- Erkennt Halluzinationen
- Erkennt Drift

## Erwartetes Verhalten
- LLM listet exakt die Module aus instructions.md
- Keine zusätzlichen Dateien
- Keine fehlenden Dateien
# Projekt-Start-Prompt

## Prompt
"Lies diese instructions.md vollständig. Welche Dateien würdest du erzeugen und in welcher Reihenfolge?"

## Warum dieser Prompt so aufgebaut ist
- Testet Strukturverständnis
- Prüft, ob das LLM die Projektarchitektur erkennt
- Erkennt Halluzinationen (erfundene Dateien)
- Erkennt Drift (abweichende Reihenfolge)

## Erwartetes Verhalten
- LLM listet exakt die Module aus instructions.md
- Keine zusätzlichen Dateien
- Keine fehlenden Dateien
